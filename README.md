## about

＊友人が考案したゲームをWebSocketを使用して実装した。

テーマについて回答した人を予想する，親睦を深めるためのレクリエーションでのゲーム

ゲーム内容：
1. あるテーマ（お題）がホストから出題される
2. テーマに参加者が自身なりの回答
3. 回答を一つづつ，誰が回答したものなのか予想して投票
4. 答え合わせをする。
